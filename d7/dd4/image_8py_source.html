7 Combinations To Account For
8 ===========================
9 
10 * Pillow:

12  * never has ``_imaging`` under any Python

14 
15 * PIL
16 
17  * CPython 2.x may have _imaging (& work)
18  * CPython 2.x may *NOT* have _imaging (broken & needs a error message)
19  * CPython 3.x doesn't work
20  * PyPy will *NOT* have _imaging (but works?)
21  * On some platforms (Homebrew and RHEL6 reported) _imaging isn't available,
22  the needed import is from PIL import _imaging (refs #21355)
23 
24 Restated, that looks like:
25 
26 * If we're on Python 2.x, it could be either Pillow or PIL:
27 
28  * If ``import _imaging`` results in ``ImportError``, either they have a
29  working Pillow installation or a broken PIL installation, so we need to
30  detect further:
31 
32  * To detect, we first ``import Image``.
33  * If ``Image`` has a ``alpha_composite`` attribute present, only Pillow
34  has this, so we assume it's working.
35  * If ``Image`` DOES NOT have a ``alpha_composite``attribute, it must be
36  PIL & is a broken (likely C compiler-less) install, which we need to
37  warn the user about.
38 
39  * If ``import _imaging`` works, it must be PIL & is a working install.
40 
41 * Python 3.x
42 
43  * If ``import Image`` works, it must be Pillow, since PIL isn't Python 3.x
44  compatible.
45 
46 * PyPy
47 
48  * If ``import _imaging`` results in ``ImportError``, it could be either
49  Pillow or PIL, both of which work without it on PyPy, so we're fine.
50 
51 
52 Approach
53 ========
54 
55 * Attempt to import ``Image``
56 
57  * ``ImportError`` - nothing is installed, toss an exception
58  * Either Pillow or the PIL is installed, so continue detecting
59 
60 * Attempt to ``hasattr(Image, 'alpha_composite')``
61 
62  * If it works, it's Pillow & working
63  * If it fails, we've got a PIL install, continue detecting
64 
65  * The only option here is that we're on Python 2.x or PyPy, of which
66  we only care about if we're on CPython.
67  * If we're on CPython, attempt to ``from PIL import _imaging`` and
68  ``import _imaging``
69
