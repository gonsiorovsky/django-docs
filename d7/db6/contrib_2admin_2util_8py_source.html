51  Ensure that primary key values do not confuse the admin URLs by escaping
52  any '/', '_' and ':' and similarly problematic characters.
53  Similar to urllib.quote, except that the quoting is slightly different so
54  that it doesn't get automatically unquoted by the Web browser.

397
