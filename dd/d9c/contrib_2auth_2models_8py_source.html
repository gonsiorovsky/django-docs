52 
53  The permission system is used by the Django admin site, but may also be
54  useful in your own code. The Django admin site uses permissions as follows:
55 
56  - The "add" permission limits the user's ability to view the "add" form
57  and add an object.
58  - The "change" permission limits a user's ability to view the change

323  """
324  Returns True if the user has the specified permission. This method

327  auth backend is assumed to have permission in general. If an object is
