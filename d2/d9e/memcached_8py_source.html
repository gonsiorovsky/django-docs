40  """
41  Memcached deals with long (> 30 days) timeouts in a special
42  way. Call this function to obtain a safe value for your timeout.
43  """

48  # Using 0 in memcache sets a non-expiring timeout.
49  return 0

52  # in memcache backends, a negative timeout must be passed.
53  timeout = -1
