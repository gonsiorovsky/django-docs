970  in Python 2.6. The important functionality here is that the server is non-

973  asked to stop.

975  can be removed as `WSGIServer` will have this ability to shutdown on
976  demand and will not require the use of the _ImprovedEvent class whose code
977  is borrowed from Python 2.7.

1011  "server might be stuck or generating a slow response.")

1075  http requests.

1077  if self.connections_override:

1079  # provided by the main thread.

1084  handler = StaticFilesHandler(_MediaFilesHandler(WSGIHandler()))

1087  # one that is free to use for the WSGI server.

1102  raise
